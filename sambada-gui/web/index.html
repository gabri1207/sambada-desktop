About SAMBADA Studio

SAMBADA is a landscape genomics software developed by the Laboratory of Geographic Information Systems (LaSIG) at EPFL (Sylvie Stucki, Stéphane Joost). It detects associations between genetic markers and environmental variables (selection signatures) and analyses spatial dependence.

SAMBADA Studio is a graphical interface that makes the software easier to use. It does not alter SAMBADA's behaviour: it prepares the parameter file and calls the official programs.

SAMBADA is distributed under the GPL v3 licence — see COPYING and AUTHORS.
